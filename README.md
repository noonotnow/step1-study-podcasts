# 🎧 Step 1 Study Podcasts

A single-page AI podcast app for USMLE Step 1 review — a companion to [study.justlikekatie.com](https://study.justlikekatie.com).

Uses the **Web Speech API** (browser text-to-speech) to read through structured podcast episodes covering all major Step 1 topics. Everything runs client-side in a single HTML file — no backend, no audio files, no separate pages.

🔗 **Launch the app:** [noonotnow.github.io/step1-study-podcasts](https://noonotnow.github.io/step1-study-podcasts/)

---

## 📚 Topics Covered

- Nutrition
- Biostatistics
- Microbiology
- Pharmacology
- Cardiology
- Endocrine
- Neurology
- Daily Review

---

## 🛠 How It Works

- **Single HTML file** (`index.html`) — fully self-contained
- **Browser TTS** via the Web Speech API — no audio files needed
- **Interactive controls** — play, pause, skip, adjust speed and voice
- **Dark theme** with Tailwind CSS, Inter font — consistent with the study hub

## ➕ Adding or Editing Content

All podcast content lives in the single `index.html` file. To update topics or add new episodes, edit the data structures within the HTML and push to `main` — GitHub Pages deploys automatically.
