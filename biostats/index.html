🎯 High-Yield Concepts

  • Sensitivity: TP / (TP + FN) — rules OUT disease (SnNout). High sensitivity = good screening test.
  • Specificity: TN / (TN + FP) — rules IN disease (SpPin). High specificity = good confirmatory test.
  • PPV/NPV: Affected by prevalence — PPV ↑ with higher prevalence; NPV ↑ with lower prevalence.
  • Relative Risk: Used in cohort studies. RR = [A/(A+B)] / [C/(C+D)].
  • Odds Ratio: Used in case-control studies. OR = (A×D) / (B×C). Approximates RR when disease is rare.
  • Type I Error (α): False positive — rejecting H₀ when it's true. p-value < 0.05.
  • Type II Error (β): False negative — failing to reject H₀ when false. Power = 1 − β.
